3.O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?

15.O que é o Flexbox no CSS e para que ele é utilizado?
R: O Flexbox é uma ferramenta do CSS que permite criar layouts flexíveis e responsivos em páginas da web

16.Qual a diferença entre justify-content e align-items em um container Flexbox?
R: Em um container Flexbox, a propriedade justify-content controla o alinhamento dos itens no eixo principal, enquanto a propriedade align-items controla o alinhamento no eixo transversal. 


17.Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
R: O flex-grow define a habilidade de um flex item de crescer, caso necessário.

18.Explique a diferença entre flex-direction: column; e flex-direction: row;.
R: A diferença entre flex-direction: column; e flex-direction: row; é a direção em que os elementos são dispostos no container. com flex-direction:row os elementos são alinhados em linha, com flex-direction:columm os elementos são alinhados em colunas
